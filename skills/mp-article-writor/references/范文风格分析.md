# 范文风格分析

从作者历史文章中提炼的写作模式和风格特征，供生成文章时参考校准。提炼方法参见 README 中「自定义范文风格分析」章节。

## 生活类范文「过年吃太好的小狗也会让人烦恼啊」

### 叙事策略

用「名侦探破案」的叙事框架讲一个狗拉肚子的生活琐事。这种「大词小用」的反差感是作者幽默的核心手法——用严肃的叙事结构承载日常的荒诞。

### 关键技法

- **具体到荒谬的细节**：不说「笼子里很脏」，而是精确描述四处喷射痕迹的位置、凝固程度、喷射时间推断。细节越精确，反差越好笑
- **角色化叙事**：狗是「犯罪嫌疑狗」，主人是「胡警官」，猫是「陪审团成员」。给日常角色赋予戏剧身份
- **emoji 点缀**：🐶🤔❗🙅‍♀️🎉 用 emoji 替代语气词，增加轻松感但不过度
- **极短段落**：几乎每段不超过 3 行，节奏极快
- **结尾温暖收场**：从搞笑回到「猫猫祝大家发大财」，荒诞归于日常

### 语言特征

- 零套话，零空话
- 用「遂」「鉴于」「当庭驳回」等文言/法律用语制造幽默
- 自然穿插图片，图片说明本身也是叙事的一部分

## 个人实践范文「你是专家这句话，到底是在帮 AI 还是在害你？」

### 叙事策略

典型的英雄之旅弧线——从一个日常好奇出发（给 AI 加专家身份到底有没有用），用实验验证，最终得出反直觉的结论。

### 结构拆解

1. **钩子**：「社交媒体上的提示词秘籍」→ 大家都在用但没人验证
2. **方法论交代**：先做文献调研（三个 AI），再设计 2×2 实验矩阵，交代清楚让读者信任
3. **最刺激的结果先放**：不按实验顺序，而是把「专家幻觉」这个最令人不安的结果放在最前
4. **交叉验证锁定变量**：用 2×2 矩阵排除混淆变量，逻辑链条清晰
5. **落地为实用指南**：从实验结论推导出「什么时候该用/不该用」
6. **终极比喻收束**：「遥控器，不是外挂」——一个比喻概括全文核心观点

### 关键技法

- **数据先行**：「120 多次 API 调用、2 个模型、5 轮实验」，用数字建立可信度
- **引用 AI 原始输出**：用 blockquote 展示 AI 的实际回答，让读者自己判断
- **反直觉发现作为转折点**：「专家身份 + 无推理能力 = 最危险的组合」
- **加粗极其克制**：只在核心论点处加粗，每个加粗都是一个可以独立成立的观点
- **附录透明度**：完整披露 AI 辅助创作过程，增强信任

### 语言特征

- 冷静但有力，不喊口号，让数据和逻辑说话
- 自我调侃穿插其中（「我决定用最笨的办法来验证」）
- 技术细节保留到「读者能复现」的程度，不多不少

## 跨文章共性模式

1. **永远从具体场景切入**：小狗拉肚子、给家人解释路由器、听到同事一句话。从来不是「在当今XX的时代」
2. **观点用加粗独立成行**：关键判断单独一行加粗，形成「扫读锚点」
3. **图片是叙事的一部分**：不是装饰，每张图都承担叙事功能
4. **结尾不说教**：用比喻或回扣开头的方式收束，不做「总结陈词」
5. **人称统一用「我」**：个人实践用第一人称，保持亲近感
6. **长短段落交错**：短段制造节奏感和重点，长段承载论证和细节
